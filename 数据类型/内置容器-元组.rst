tuple([iterable]) -> new tuple
==============================
    任意无符号的对象，以逗号隔开，默认为元组

    .. code-block:: python

        t = 1, 2, 3
        def demo(): return 1, 2, 3
        print(t)       #: (1, 2, 3)
        print(demo())  #: (1, 2, 3)
