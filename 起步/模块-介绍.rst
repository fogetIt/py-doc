##### Anaconda
- 一个`python`的发行版，包括了`python`和很多常见的软件库
    + Python
    + [ipython](http://archive.ipython.org/release/2.3.1/ipython-2.3.1.zip)
        *  解压，进入解压包目录
        *  python setup.py install
        *  ipython在Scripts文件夹下
    + Ipython QTConsole
    + IPython Notebook
    + Spyder(IDE)

pip/conda（anaconda自带的包管理器） list，查看已经安装的包。
pip/conda install 包名，安装新的包。
